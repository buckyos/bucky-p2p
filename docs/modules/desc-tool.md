# desc-tool

## 类型
- 工具型二进制和辅助库

## 职责
- 创建、检查、修改、计算并签名与 descriptor 相关的制品。

## 关键边界
- 范围内：
  - `desc-tool/src/*.rs`
  - 生成对象和签名文件的处理逻辑

## 依赖
- CYFS object and crypto crates

## 模块级别
- Tier 2：相对独立的操作工具，但输出具备安全敏感性

## 必需的验证倾向
- CLI 行为、文件格式处理和签名流程需要 unit 覆盖，并至少提供一个面向操作员的 DV 命令。
- 即使代码 diff 很小，安全敏感的输出变更也需要评审额外关注。
