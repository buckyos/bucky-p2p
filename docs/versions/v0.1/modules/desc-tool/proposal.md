# desc-tool 提案

## 背景与目标
- `desc-tool` 是工作区中用于 descriptor 创建、检查、修改、计算和签名的工具。
- 目标是让安全敏感的 CLI 和文件输出变更可审计，并受到阶段门禁控制。

## 范围
### 范围内
- CLI 子命令
- descriptor 文件处理和转换
- 签名与计算行为

### 范围外
- 拥有运行时服务行为
- 替换底层 CYFS 对象语义

### 与相邻模块的边界
- `desc-tool` 只负责面向操作员的文件处理行为。

## 约束
- 生成的制品属于安全敏感输出
- 行为变更需要显式的面向操作员 DV
- 格式或签名变更需要更强的评审关注

## 高层结果
- 工具行为变更变得可评审、可复现
- 文件输出语义不再静默漂移

## 风险
- 输出制品损坏或不兼容
- 签名回归
- 被很小的 CLI diff 掩盖的隐藏行为变化
